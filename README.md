## Too Doo Web Development

Too doo is a organization tracking app. It allows users to create tasks, add it to a to do list, as well as use pre made tasks given by the website. This allows the user to add tasks and cross them off as they complete them

### Personal Accounts
Users can create an account and log in on the website. The website however can be used by guests for searching and browsing, however, they cannot have access to the premium features such as :
- Creating tasks
- Favouriting tasks
- Creating to do lists
